# Li
基于movielens电影推荐和网页可视化（附海报数据集）
首先 代码来自于 飞浆 https://www.paddlepaddle.org.cn/tutorials/projectdetail/3833063
在以上代码的基础上进行海报网页可视化

注意！！！！！
给出的poster电影海报不全 去 https://github.com/11Li11/movie_poster.git 下载完整poster海报数据集进行替换即可
